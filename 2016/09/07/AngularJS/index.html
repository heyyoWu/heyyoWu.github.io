##AngularJS

###指令

ng-app 定义根元素(网页加载完毕会自动初始化应用程序)
ng-init 赋初始值(使用控制器模块来替代)？？？
ng-model 绑定HTML元素到应用程序数据

  1. 为应用程序数据提供验证（number 、email、required）
  2. 位应用程序数据提供状态（invalid、dirty、touched、error）
  3. 位HTML元素提供CSS类
  4. 绑定HTML元素到HTML表单

ng-repeat 对于集合中的每个项会克隆一次HTML元素(循环)
自定义指令：使用.directiive来创建、指令命名(驼峰)、使用时需要使用分割线-

  2. 属性 （A）

  3. 类名 （C）

  4. 注释 （M）

  5. 使用restrict限制访问方式

##JS模式

####1. MVC

  • View 视图— 用户界面
  • Controller 控制器—业务逻辑
  • Model 模型—数据保存
  1. View传送指令到Controller
  2. Controller完成指令后要求Model改变状态
  3. Model将新的数据发送到View，用户得到反馈
  • 总结：所有通信都是单向的

####2. MVP

  1. Controller改名为Presenter
  2. View与Model不发生关系，都通过Presenter传递
  3. View非常薄，不部署任何逻辑(被动视图)，没有主动性
  4. Presenter非常厚，所有的逻辑都在这
    总结：各部分之间的通信都是双向的

3. MVVM

  1. View和ViewModel（Presenter）的双向绑定，比如AngularJS
